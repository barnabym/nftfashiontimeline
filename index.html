This website is about the emerging Metaverse. The focus will be on 'verses' (parts of the Metaverse) that are open worlds where users can earn money. It will also acknowledges that things like Discord are also part of the Metaverse.

2021 - coming

  • 2021 - Bitcountry - polkadot virtual world
  • 2021 - Fragcolor - metaverse 3d model prefabs, introduces the create to earn economy
  • 2021 - Epic Games - 1 billion funding for Metaverse vision
  • 2021 - Facebook - announces it is becoming a Metaverse company
  • 2021 - Wilder World - A virtual world set up by a DAO. Frank Wilder was the artist who started it. Source 2
  • 2021 - Flare Network Metaverse - people behind XRP virtual world
  • 2021 - Nifty Island - is an open social game hub and gallery that harnesses nfts and crypto to make the metaverse a reality
  • 2021 - rp1 - scalable Metaverse software
  • 2021 - Realm - "personalised metaverses"
  • 2021 - Genies - avatar creator for verses
  • 2021 - Crucible - Metaverse avatar software and more
  • 2021 - Blankos open-world multiplayer game with avatars as NFTs
  • 2021 - V-Market - 6th VR Marketplace event

  • 2021 - IPSME - is introduced as a solution for integrating highly disparate systems

  • 2021 - ReadyPlayerMe Wolf3d avatars for the Metaverse
  • 2021 - Meebits seed avatars for the Metaverse
  • 2021 - webaverse.com Webaverse
  • April 2021 - oncyber.io - 3d gallery NFT platform
  • 2021 - OVR - AR platform where can buy land as NFTs
  • 2021 - Substrata - verse for graphics geeks
  • 2021 - Superworld - AR virtual world where can buy land as NFTs
  • 2021 - OVR - AR Spatial web platform for live events. Has virtual land as NFTs

1996

1995

  • 1995 - Worlds, Inc. - first 3d non game based social virtual world

1994

  • 1994 - Web World - 2.5D (isometric) world with first open-ended building capability1

1986

  • 1986 - Habitat for the Commodore 64 - first virtual world with a 2d graphical interface. Also, first virtual world to use the term 'avatar'1

1970
